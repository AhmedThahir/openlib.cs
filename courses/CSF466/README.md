# Service Oriented Computing

## Overview

Course covers `XML` and `Web Service Principles` & technology. XML topics include `Signatures`, `Encryption` and `WS security`. Other topics include `PKI`, `SAML`, `Ajax`, `REST` and `SOAP` services.
